Key Design Patterns for Large Dialogs:
  • Sticky Header: Title and entity info stay visible when scrolling content
  • Sticky Footer: Action buttons (Save, Delete, Cancel) remain accessible
  • Scrollable Middle: All form sections (Properties, Attributes) scroll independently
  • Visual Sections: Grouped form fields with clear spacing and labels
  • Touch-Friendly Buttons: Minimum 44px height for mobile tap targets
  • Icon Badges: Attribute types are clearly labeled (number, string, array, etc.)
5. Left Panel on Mobile: Hidden by Default, Accessible via UI

On mobile, the left panel (ViewManager, EntityPalette, StaticComponentPalette) is hidden to maximize canvas space. Access it via header dropdown and floating action button.
